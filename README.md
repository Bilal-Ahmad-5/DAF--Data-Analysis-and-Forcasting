# 📊 DAF — Data Analysis & Forecasting

> *Turn raw data into clear insights and confident predictions — all inside Google Colab.*

A Google Colab notebook for analyzing data and building forecasting models — from raw data to results, in one place.

---


## Quick Start

1. Open the notebook in **Google Colab**
2. Upload your data file or connect **Google Drive**
3. Hit `Runtime → Run all` and let it flow

That's it. No setup. No installs. Just results.

---

##  What's Inside

| Step | What Happens |
|------|-------------|
|  **Load** | Import data from CSV, Excel, or Google Drive |
|  **Clean** | Handle missing values, fix types, scale features |
|  **Explore** | Stats, distributions, correlations, and charts |
|  **Feature Engineering** | Lag windows, date decomposition, encoding |
|  **Train** | Cross-validation, tuning, and forecasting models |
|  **Evaluate** | Metrics, residual plots, and model comparisons |
|  **Export** | Save predictions and model files back to Drive |

---

## Requirements

**None locally** — the notebook runs entirely in Google Colab.

Any additional packages are installed automatically at the top of the notebook.

---

## Outputs

-  Model predictions exported as **CSV**
-  Trained model saved as a **pickle file** to Google Drive
-  Charts and diagnostics rendered **inline**
